Jumping into Geometry!

This site is an introduction into 2D shapes and 3D figures.

The purpose of this website is to introduce parents and students to the basics of geometry. This includes identifying (naming) 2D shapes and 3D figures, understanding the difference between 2D and 3D figures, as well as naming attributes of both in order to identify them. This website will include not only information regarding the concepts but also provide resources for parents to help their child at home. I want a website that is colorful yet clean that is interactive. This will help it appeal to children and parents.


Goals of this site...

  1. Students will be able to name 2D shapes.
  2. Students will be able to name 3D figures.
  3. Students will be able to tell the difference between 2D and 3D figures due to the differences in attributes.

Objective of this site:

  • Clean & visually appealing
  • Directed towards young learners
  • Provide resources for families
  • Teach introductory basics of geometry
